 -- Aula 3 - Vídeo 9 - capítulo 3 PDF
A HTML as CSS
HTML(HyperText Markup Language) e CSS(Cascading Style Sheets(folhas de estilo em cascata)) são linguagens, mas não são linguagens de programação

HTML é focada em conteúdo (textos, imagens, vídeos, listas...)
CSS é focada é design (cores, fontes, sombras, posicionamento...)

<> tags ou colchete angular

<h1> = abertura de tag
Exemplo de título = conteúdo
</h1> = fechamento de tag

Estrutura básica obrigatória de HTML

	<!DOCTYPE html>
	<html lang="pt-br">
		<head>
		//área de configurações
			<meta charset="UTF-8">
			<meta name="viewport"content="width=device-width, initial-scale=1.0">
			<title>Document</title>
			
		</head>
		<body>
		//corpo do site
			<h1>Olá, Mundo!</h1>
		
		</body>

	</html>
